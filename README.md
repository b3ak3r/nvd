# Nvd

Automate the construction of your own vulnerability database, based on NVD

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'nvd'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nvd

## Usage

    $ vagrant up
    $ vagrant ssh
